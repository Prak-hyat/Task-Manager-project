# Smart Task Manager  
A Python application that allows users to create, manage, complete, and delete tasks using an object-oriented design.  
The program includes multiple classes, a GUI built with Tkinter, and automated tests using pytest.

---

## Features
- Add tasks with title and description  
- Mark tasks as completed  
- Delete all completed tasks at once  
- Display all, pending, or completed tasks  
- Save and load tasks using JSON storage  
- Supports UrgentTask and TimedTask subclasses  
- Pytest test suite included
